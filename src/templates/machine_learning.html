Unsupervised Machine Learning

Clustering of Theft Related Crimes

This project used a clustering Machine Learning algorithm to group crime across multiple different parameters. Clustering is a type of unsupervised learning that groups data points together. This group of data points is called a cluster. Cluster modeling is a form of unsupervised machine learning. The model that is created finds structure from the data input into the model, as opposed to being given an outright structure. This type of machine learning is excellent at visualizing and discerning patterns that may be hidden within the dataset.

The models that were built to represent these datasets were specifically built on a K-means algorithm, and then a Principal Component Analysis (PCA) was was applied to the model to reduce some of the complex dementionality. The "K-means" algorithm will cluster the dataset into K clusters based on their proximity within the dataset to a centroid point for that cluster. In order to determine the optimal number of clusters that the K-means algorithm was to produce for each dataset's model, we ran an inertia test; an elbow curve test. Below is the result of the inertia test from the Theft dataset and K-means model.

Because this dataset looked at many datapoints, and the dataset didn't offer a known structure to compare the data, we then applied the PCA statistical technique. This is a method of reducing the number of features to compare by the clustering model. This allows us to visualize the clustered models across the entire dataset, with only three dimensions, while still keeping as much of the original features and data as possible within these clusters.

We used the results of these clusters to generate questions and to explore our datasets in new ways. We examined the outcome of these models, and began to ask questions like; "maybe there is a connection between time fo day and the type of crime of crime that was commited", or "there may be a trend between the location the crime was committed and the weather"...
Then using these visuals and our exploration time we generated a story of the crime that was comitted within the greater Austin metroplex (2020-2021), these can be seen on the other pages within this application.
